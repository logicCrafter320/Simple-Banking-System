# Simple-Banking-System
A Java-based banking system project developed as part of academic work

This project demonstrates Object-Oriented Programming concepts and provides basic banking functionalities for customers and managers.

---

## 🚀 Features

### 👤 Customer Functions
- Create new account  
- Deposit money  
- Withdraw money  
- Check balance  
- View transaction history  

### 🛠 Manager Functions
- View all accounts  
- Search for customers  
- Manage accounts
