📌 Simple Banking System

A Java-based console banking application demonstrating core Object-Oriented Programming (OOP) concepts.
This project simulates basic banking operations for customers and bank managers, making it suitable for academic learning or introductory software development projects.

🚀 Features
👤 Customer Functions

Create a new account

Deposit money

Withdraw money

Check account balance

View transaction history

🛠️ Manager Functions

View all accounts

Search for customers

Manage customer accounts

Deactivate accounts

View all transactions

🏗️ Tech Stack

Java (Core Java)

Object-Oriented Programming (OOP)

File handling / Collections (depending on your implementation)

Console-based user interface

🔍 OOP Concepts Used

Encapsulation – private fields with public getters/setters

Abstraction – separate classes for customer, account, manager

Inheritance – extend behavior (if implemented)

Polymorphism – method overriding (if implemented)

Object relationships – Customer ↔ Account ↔ Transactions
